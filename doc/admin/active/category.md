#### 列表接口

> 获取案例数据

+ __接口地址__： __/vote/category/list__

+ __请求参数__

|  参数名称  | 参数类型 | 参数说明 |
| --------- | -------- | ------- |
| page | int | 当前页 默认1 |



+ __响应参数__

|  参数名称  | 参数类型 | 参数说明 |
| --------- | -------- | ------- |
| pages | Object | 总数据， 用于分页 |
| list | Object | 数据相关 |

>  pages 类型：__[Object]__

|  参数名称  | 参数类型 | 参数说明 |
| --------- | -------- | ------- |
| total | number | 总数据量  |
| pagesize | number |  每页数据 |

>  list 类型：__[Object]__

|  参数名称  | 参数类型 | 参数说明 |
| --------- | -------- | ------- |
| id | int(11) | id编号 |
| c_name | varchar(255) | 分类名 |
| pname | int(11) | 父级分类名 |
| update_time | int(11) |更新时间  |
| create_time | int(11) | 创建时间 |
| status | tinyint(2) | 状态 |
