# 用户相关

----------------
### 用户登录

+ __接口地址__： __/System/User/login__

+ __请求参数__

|  参数名称  | 参数类型 | 参数说明 |
| --------- | -------- | ------- |
| *user | string | 用户名称 |
| *pwd | string | 用户密码 |
| *code | string | 验证码  |

+ __响应参数__

>   param 类型：__[Object]__
